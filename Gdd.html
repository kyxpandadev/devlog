Game Design Document

  • Game name: G'old Days
  • Made by Kyxpanda Dev
  • Engine used: Unreal Engine 5.3
  • Programming Style: Blueprint + C++

Gameplay overview

Looter shooter inspired by the likes of Destiny, warframe set in a Futuristic wild west. You are in a constant dangerous cat and mouse game with the law and different gangs. Loot weapons and materials to get on top of the threats you will face

Narrative

Set in a more futuristic Wild west setting Outlaws are everywhere and trying to overtake Each other in the amount of wealth they each hold. its like a game. You play as one of these outlaws starting from nothing
